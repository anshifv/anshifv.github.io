What is Nmap?

📅 Published: April 2026 • 📖 3 min read

Nmap (Network Mapper) is an open-source tool used for network discovery and security auditing. It was created by Gordon Lyon and is considered the "Swiss Army knife" of networking. Ethical hackers and system administrators use Nmap to identify devices running on a network, discover open ports, and detect security risks.

How Nmap Works

Nmap works by sending specially crafted packets to target hosts and analyzing the responses. By examining how a system responds to different types of probes, Nmap can determine which ports are open, what services are running, what operating system is being used, and even what version of software is listening on a port.

# Basic scan of a single host
nmap 192.168.1.1

# Scan specific ports
nmap -p 22,80,443 192.168.1.1

Common Uses of Nmap

Penetration testers use Nmap during the reconnaissance phase to map out a target's attack surface. System administrators use it for inventory management and to verify security policies. It can detect unauthorized devices on a network, identify outdated software versions, and help harden systems by showing what services are exposed to the internet.
